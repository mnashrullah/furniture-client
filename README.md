# Furniture-client

Check [server](https://github.com/mnashrullah/furniture-server)

### Install nodejs
Need [Node.js](https://nodejs.org/) to run.

### Install ionic
Need ionic framework
```sh
$ npm install -g ionic@latest
```

### Install package

```sh
$ npm install
```

### Running application

Make sure server is running and database already imported.

```sh
$ ionic serve
```
