# Furniture-client

Check [server](https://github.com/mnashrullah/furniture-server)

### Installation

Need [Node.js](https://nodejs.org/) to run.


Need ionic framework
```sh
$ npm install -g ionic
```


Install the dependencies

```sh
$ npm install
```

running client using ionic
```sh
$ ionic serve
```
